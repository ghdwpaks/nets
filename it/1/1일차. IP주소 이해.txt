▶ Classful address

IPv4=(?).x.x.x                     Subnet Mask (1=Network자리, 0=Host자리)
----------------         -------------------------------------------------
A Class : 1 - 126                255./0.0.0          <예> 126./0.255.100
B Class : 128 - 191            255.255./0.0      <예> 191.0./255.100
C Class : 192 - 223            255.255.255./0  <예> 223.0.255./100
※ 127.0.0.1= loopback 주소 (자기 자신)
----------------------------------------------------------------------
D Class : 224-239, Multicast주소로 각각 예약(그룹)  <예> 224.0.0.10 = 인사부(컴퓨터10대)
E Class : 240 - 255, 연구용


▶ IPv4 대화방법
. unicast = 1:1
. multicast = 1:Group
. broadcast = 1:다
